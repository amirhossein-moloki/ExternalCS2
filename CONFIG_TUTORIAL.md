# راهنمای جامع تنظیمات (Detailed Configuration Guide) 📘

این راهنما تمام بخش‌های موجود در فایل `config.json` را با جزئیات کامل، مقادیر مجاز و توضیحات فارسی توضیح می‌دهد.

---

## 🔝 بخش اصلی (Top-level Settings)

این تنظیمات کلیدهای اصلی و رفتارهای کلی برنامه را کنترل می‌کنند.

| کلید (Key) | مقدار پیش‌فرض | بازه/مقادیر مجاز | توضیح فارسی |
| :--- | :--- | :--- | :--- |
| `aimBot` | `true` | `true`, `false` | فعال/غیرفعال کردن کلی سیستم هدف‌گیری خودکار. |
| `aimBotAutoShoot` | `true` | `true`, `false` | شلیک خودکار بلافاصله پس از قفل شدن روی هدف. |
| `bombTimer` | `true` | `true`, `false` | نمایش زمان باقی‌مانده تا انفجار بمب روی صفحه. |
| `skeletonEsp` | `true` | `true`, `false` | نمایش اسکلت‌بندی دشمنان از پشت دیوار. |
| `skeletonShowHeadCircle` | `true` | `true`, `false` | نمایش دایره دور سر در حالت اسکلتون. |
| `triggerBot` | `true` | `true`, `false` | شلیک خودکار وقتی دشمن دقیقاً روی نشانه قرار می‌گیرد. |
| `aimBotKey` | `1` | کد کلید (1-255) | کلید فعال‌ساز ایم‌بات (مثلاً 5 برای Mouse 4). |
| `triggerBotKey` | `164` | کد کلید | کلید فعال‌ساز تریکربات (پیش‌فرض Alt چپ). |
| `menuToggleKey` | `45` | کد کلید | کلید باز/بسته کردن منوی تنظیمات (پیش‌فرض Insert). |
| `teamCheck` | `true` | `true`, `false` | اگر فعال باشد، روی هم‌تیمی‌ها قفل نمی‌شود. |

## 📦 تنظیمات باکس (ESP Box Config)
در مسیر `esp.box` قرار دارد.

| کلید | مقدار پیش‌فرض | توضیح فارسی |
| :--- | :--- | :--- |
| `enabled` | `true` | فعال‌سازی کلی باکس دور دشمن. |
| `showBox` | `true` | نمایش مستطیل دور دشمن. |
| `showName` | `true` | نمایش نام بازیکن. |
| `showHealthBar` | `true` | نمایش نوار سلامتی (جان). |
| `showHealthText` | `true` | نمایش عدد سلامتی. |
| `healthPosition` | `0` | موقعیت نوار سلامتی (0: چپ، 1: بالا، 2: راست، 3: پایین). |
| `showDistance` | `true` | نمایش فاصله تا هدف بر حسب متر. |
| `showWeaponIcon` | `true` | نمایش آیکون اسلحه دشمن. |
| `showArmor` | `true` | نمایش مقدار زره و داشتن کلاهخود. |
| `showVisibilityIndicator`| `true` | تغییر رنگ یا علامت برای دشمنانی که در دید هستند. |
| `enemyColor` | `"FF8B0000"` | رنگ دشمن (قالب ARGB - هگزادسیمال). |
| `visibleAlpha` | `"FF"` | شفافیت دشمن وقتی در دید است (00 تا FF). |
| `invisibleAlpha` | `"88"` | شفافیت دشمن وقتی پشت دیوار است. |

## 📡 رادار (ESP Radar Config)
در مسیر `esp.radar` قرار دارد.

| کلید | مقدار پیش‌فرض | بازه/توضیح |
| :--- | :--- | :--- |
| `enabled` | `true` | فعال‌سازی رادار کوچک روی صفحه. |
| `size` | `150` | اندازه رادار به پیکسل (مثلاً 100 تا 400). |
| `x`, `y` | `50`, `50` | مختصات قرارگیری رادار روی صفحه. |
| `maxDistance` | `100.0` | حداکثر فاصه‌ای که در رادار نمایش داده می‌شود. |
| `showLocalPlayer` | `true` | نمایش خودتان در مرکز رادار. |

---

## 🎯 تنظیمات نشانه‌گر و FOV (AimCrosshair Config)
در مسیر `esp.aimCrosshair` قرار دارد.

| کلید | مقدار پیش‌فرض | توضیح فارسی |
| :--- | :--- | :--- |
| `enabled` | `true` | نمایش یک نشانه‌گر کمکی که با لگد اسلحه جابجا می‌شود. |
| `radius` | `6` | اندازه نشانه‌گر. |
| `recoilScale` | `2.0` | شدت جابجایی نشانه‌گر بر اساس لگد اسلحه. |
| `showFovCircle` | `false` | نمایش دایره‌ای که محدوده عملکرد ایم‌بات را نشان می‌دهد. |
| `fovCircleRadius` | `120` | شعاع دایره FOV به پیکسل. |

---

## 🎹 تنظیمات دقیق ایم‌بات (AimBot Tuning)
در مسیر `aimBotTuning` برای حرفه‌ای‌ها.

| کلید | مقدار پیش‌فرض | توضیح فارسی |
| :--- | :--- | :--- |
| `aimSmoothing` | `3.0` | نرمی حرکت (اعداد بالاتر = حرکت انسانی‌تر و کندتر). |
| `humanReactThreshold` | `30.0` | آستانه حساسیت به حرکت ماوس توسط خود شما. |
| `minShootIntervalMs` | `100` | حداقل زمان بین هر شلیک در حالت خودکار (میلی‌ثانیه). |
| `aimUpdateIntervalMs` | `500` | سرعت آپدیت منطق هوشمند ایم‌بات. |

---

## 🔊 صدا و متن اصابت (Hit Sound Config)
در مسیر `hitSound` قرار دارد.

| کلید | مقدار پیش‌فرض | توضیح فارسی |
| :--- | :--- | :--- |
| `enabled` | `true` | پخش صدا و نمایش متن هنگام تیر خوردن دشمن. |
| `hitText` | `"HIT"` | متنی که هنگام تیر خوردن نمایش داده می‌شود. |
| `headshotText` | `"HEADSHOT"` | متنی که هنگام هدشات نمایش داده می‌شود. |
| `textDurationSeconds` | `1.5` | مدت زمان باقی ماندن متن روی صفحه. |

## ⌨️ لیست کدهای کلید (Key Codes Reference)

برای تنظیم کلیدها از این اعداد استفاده کنید:

- **1**: کلیک چپ ماوس
- **2**: کلیک راست ماوس
- **4**: کلیک وسط ماوس
- **5**: دکمه جانبی ماوس (Back/Mouse 4)
- **6**: دکمه جانبی ماوس (Forward/Mouse 5)
- **16**: Shift
- **17**: Control
- **18**: Alt
- **20**: Caps Lock
- **32**: Space
- **45**: Insert
- **46**: Delete
- **65 تا 90**: حروف A تا Z

## 💡 نکات مهم
*   **فرمت رنگ‌ها:** رنگ‌ها به صورت `AARRGGBB` هستند. مثلاً `FFFF0000` قرمز خالص و غیرشفاف است.
*   **ذخیره‌سازی:** اگر تنظیمات را از داخل منوی بازی (Insert) تغییر دهید، به صورت خودکار ذخیره می‌شوند.
*   **خطا در فایل:** اگر فایل `config.json` خراب شود، برنامه آن را با تنظیمات پیش‌فرض جایگزین می‌کند.
