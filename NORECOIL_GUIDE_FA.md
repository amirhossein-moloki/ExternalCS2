# راهنمای استفاده از قابلیت No Recoil (حذف لگد اسلحه)

این مستند برای توضیح نحوه عملکرد و تنظیمات مربوط به سیستم‌های کنترل لگد در CS2GameHelper تهیه شده است.

## ۱. تفاوت بین No Recoil و RCS

در این برنامه دو روش برای مقابله با لگد (Recoil) وجود دارد:

*   **RCS (Recoil Control System - Pattern):** این سیستم با شبیه‌سازی حرکت موس به سمت پایین، لگد اسلحه را کنترل می‌کند. این روش کاملاً مشابه حرکت دست یک بازیکن حرفه‌ای است و امنیت بالاتری دارد.
*   **No Recoil (Memory):** این قابلیت مستقیماً کدهای بازی را در حافظه (RAM) تغییر می‌دهد تا اسلحه اصلاً تکان نخورد (Perfect No Recoil) یا لرزش صفحه حذف شود (No Visual Recoil).

## ۲. پیش‌نیازها

برای اینکه قابلیت **No Recoil (Memory)** به درستی کار کند، موارد زیر الزامی است:

1.  **دسترسی Administrator:** برنامه حتماً باید با کلیک راست و انتخاب `Run as Administrator` اجرا شود.
2.  **درایور DiskHelper (اختیاری اما توصیه شده):** برای امنیت حداکثری و جلوگیری از شناسایی توسط آنتی‌چیت، توصیه می‌شود از درایور `DiskHelper.sys` استفاده کنید. با این حال، در نسخه جدید، اگر درایور لود نشده باشد، برنامه سعی می‌کند از روش‌های استاندارد ویندوز استفاده کند.

## ۳. تنظیمات (config.json)

برای فعال‌سازی این قابلیت، بخش مربوطه در فایل `config.json` باید به شکل زیر باشد:

```json
"noRecoil": {
  "perfectNoRecoil": true,
  "noVisualRecoil": true
}
```

*   `perfectNoRecoil`: لگد فیزیکی گلوله‌ها را حذف می‌کند (گلوله‌ها دقیقاً وسط هدف می‌خورند).
*   `noVisualRecoil`: لرزش دوربین و صفحه هنگام تیراندازی را حذف می‌کند.

## ۴. نکات مهم

*   **تداخل:** اگر از `No Recoil (Memory)` استفاده می‌کنید، پیشنهاد می‌شود `RCS` را غیرفعال کنید تا موس شما بی‌دلیل به سمت پایین حرکت نکند.
*   **امنیت:** تغییر مستقیم حافظه بازی (Perfect No Recoil) ریسک شناسایی (Ban) بیشتری نسبت به RCS دارد. اگر اکانت شما برایتان مهم است، استفاده از RCS اولویت دارد.
*   **عدم عملکرد:** اگر با وجود تنظیمات بالا باز هم لگد حذف نشد، مطمئن شوید که آنتی‌ویروس شما مانع از دسترسی برنامه به حافظه بازی نشده است.
